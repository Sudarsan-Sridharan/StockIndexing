

### Project Structure 

```/
   |-- src -> source for backend spring boot server.
   |-- client -> source for frontend React application, with embedded developement server.
```

### Run
	Inside /client directory, run ``` npm build ```
	In root directory, run ``` mvn spring-boot:run ````

### Technologies Used
```
	
	Spring Boot
	Hibernate
	H2 Database
	JCS
	Websockets

	HTML5
	Bootstrap
	ReactJS
	Webpack

```
